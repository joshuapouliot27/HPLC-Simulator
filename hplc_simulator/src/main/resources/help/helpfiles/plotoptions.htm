Plot Options panel

The Plot Options panel contains several controls that display a second plot (the y-axis will be on the right side of the chromatogram display) of one of the following relationships:

  • No plot - Removes the second plot
  • Solvent B fraction - Displays the solvent B fraction vs. time relationship.
  • Backpressure - Displays the backpressure vs. time relationship.
  • Mobile phase viscosity - Displays the mobile phase viscosity vs. time relationship.
  • Retention factor of: - Displays the retention factor vs. time relationship of the selected compound
  • Position along column of: - Displays the position of the selected compound along the column (0 mm = column inlet) as a function of time.
