Column Properties panel

The Column Properties panel contains the following items:

  • Stationary phase control
  • Column length control
  • Column diameter control
  • Particle size control
  • Interparticle porosity control
  • Intraparticle porosity control
  • Total porosity indicator
  • Void volume control
  • Void time control
  • Van Deemter parameter controls
  • Reduced plate height control
