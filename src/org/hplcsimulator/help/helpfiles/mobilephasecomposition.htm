Mobile Phase Composition panel

The Mobile Phase Composition panel contains the following items:

  • Solvent A control
  • Solvent B control
  • Isocratic elution mode control
  • Gradient elution mode control

If isocratic elution mode is selected:

  • Solvent B fraction control

If gradient elution mode is selected:

  • Gradient program table
  • Mixing volume control
  • Non-mixing volume control
  • Total dwell volume indicator
  • Dwell time indicator
